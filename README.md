# ML Notes

ML Notes is a compact coursework repository for machine-learning practice and rendered assignment deliverables. The current project set contains a car-data assignment notebook and its exported HTML report.

## Contents

| Path | Description |
| --- | --- |
| `Desktop/Assignment 1/Assignment1_Car.ipynb` | Jupyter notebook for the first machine-learning assignment |
| `Desktop/Assignment 1/NikumbhMehul.html` | Rendered HTML submission/export for the same assignment |

## How To Use

Clone the repository and open the notebook in JupyterLab, VS Code, or another notebook environment:

```bash
git clone https://github.com/mehulnikumbh19/ML_Notes.git
cd ML_Notes
```

If you want to inspect the completed assignment without running code, open the HTML export in a browser.

## Suggested Local Setup

```bash
python -m venv .venv
. .venv/Scripts/activate  # Windows PowerShell users can run: .\.venv\Scripts\Activate.ps1
pip install jupyter pandas numpy matplotlib scikit-learn
jupyter lab
```
